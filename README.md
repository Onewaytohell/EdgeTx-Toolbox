# EdgeTx-Toolbox
Little handy tools to make life easier with EdgeTX

Switch Remapper: Useful when transferring models between radios with different switch layouts e.g A Radiomaster TX15 to TX16. Load your model file, switch remapping rules and save the updated file — no manual editing required. It will use placeholder to avoid double-remapping.

Tool for reordering screen layouts in EdgeTX model files .yml, Load your model, arrange your screens into any order using the up/down buttons and save. No manual editing of model file required.

Backup your models to a chosen directory, automatically renames models01/models02 to their actual model names for easy single-model restores, keeps raw backup copies for full radio/config restores, remembers your backup paths, and includes a dual-transmitter version with separate paths for each radio.
